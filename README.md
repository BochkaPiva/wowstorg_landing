# WOWSTORG Landing

Premium landing page for the Russian event agency ВАУСТОРГ.

## Current Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lenis
- Lucide React

## Project Structure

```text
src/app                 Application composition
src/widgets             Landing sections
src/features            Future workflows
src/entities            Domain models
src/shared              UI, config, styles, utilities
src/integrations        Future external integrations
docs                    Architecture and product notes
```

## Local Scripts

```powershell
npm.cmd run dev
npm.cmd run build
```
